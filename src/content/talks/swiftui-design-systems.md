In this talk, we'll explore how to build a design system using SwiftUI that can be shared across multiple iOS applications. Drawing from experience building components used by millions of users at Capital One, I'll share practical patterns and lessons learned.

## Topics Covered

- Component architecture and API design
- Theming and customization strategies
- Accessibility considerations
- Testing reusable components
- Documentation and developer experience
