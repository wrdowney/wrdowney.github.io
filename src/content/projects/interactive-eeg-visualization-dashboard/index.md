Created an interactive dashboard for visualizing EEG activity using React,
FastAPI, Postgres, Docker, and AWS. I also designed the data pipeline and
machine learning workflow behind it, reaching 76.75% accuracy across four key
activity channels and using GPT-4 to help summarize results for users.
